# AI, Machine Learning & MCP Frameworks Collection

This repository features a collection of simple yet powerful AI, automation projects, and Model Context Protocol (MCP) framework implementations. It is organized into two main categories: `bots` for simple automation and code generation bots, and `MCP_Frameworks` for more comprehensive MCP server integrations.

### 1. Simple Bots (`/bots`)

A collection of Python-based bots for messaging and code generation:

- **[WhatsApp Text Bot](./bots/whatsappbot)**
  A simple bot built using Python (`pywhatkit`) to send WhatsApp messages automatically. It can send text messages at scheduled times, making it useful for sending reminders, alerts, or automated messages.

- **[Python Code Generation Chatbot](./bots/simple-codebot)**
  A conversational chatbot built using the Hugging Face Transformers library that can generate Python code based on user input. Whether you need a function for data processing or a small script, this bot can generate Python code snippets in response to your queries.

### 2. MCP Frameworks Collection (`/MCP_Frameworks`)

A comprehensive collection of Model Context Protocol (MCP) servers and integrations showcasing different deployment patterns, technologies, and use cases:

- **[AI-HR Assistant](./MCP_Frameworks/ai-hr)**: PostgreSQL + Gmail + Obsidian Integration demonstrating multi-server orchestration.
- **[ClaudeOPS](./MCP_Frameworks/claude-ops)**: System Administration & DevOps using Docker and Python for secure local command execution.
- **[Remote Railway Server](./MCP_Frameworks/remote-railway-server)**: A knowledge base server deployed to the cloud using TypeScript and SQLite.
- **[Claude News](./MCP_Frameworks/claude-news)**: Multi-protocol info suite combining SSE and stdio for weather, crypto, and news.
- **[Ollama + LocalOps](./MCP_Frameworks/ollama-localops)**: Local AI models and operations setup.

*See the [MCP Frameworks README](./MCP_Frameworks/README.md) for more detailed information and setup guides for the MCP servers.*

### Technologies Used:
- **Bot Libraries**: `pywhatkit`, `transformers`, `tkinter`, `torch`, `openai`, `Python 3.x`
- **MCP Frameworks**: `Docker`, `PostgreSQL`, `SQLite`, `TypeScript`, `Node.js`, `Express.js`, `Railway`
